# EZ-ECG
CMPE 127 - Group Lab Project

## Pin Configuration
TM4C123G Board to AD8232 Board
PE2 -> Output
PD6 -> LO-
PD7 -> LO+

### Serial Pins
PA0 -> UART U0RX
PA1 -> UART U0TX
